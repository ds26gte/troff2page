9. GNU Info docs

You can call the macro

    .troff2info

within your document to signal to troff2page that it should also create an Info document.

If the basename of the input troff document is jobname, the Info file is created in jobname.info, which may be displayed on any Info viewer.

Since troff2page doesnâ€™t accept options, a good place to set .troff2info external to the input troff document is in the .troff2pagerc.tmac file, described on page 2.
